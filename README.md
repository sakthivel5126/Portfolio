# Portfolio
A modern, responsive personal portfolio website built with React and Vite. It showcases my education, skills, projects, training, certifications, resume, and contact information.

Live Demo
https://portfolio-delta-sepia-0mt10qemll.vercel.app/

Features
Landing / About — Hero section with profile introduction and bio
Skills — Technical skills grouped by category
Education — Academic background with aligned cards
Training & Certifications — In-plant training timeline and certificate viewer
Projects — Project cards with GitHub links
Resume — In-browser resume preview (PDF or built-in fallback)
Contact — EmailJS-powered contact form
Side scroll indicator — Draggable dot progress bar for page navigation
Responsive design — Works on desktop and mobile
