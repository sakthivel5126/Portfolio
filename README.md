# Premium Developer Portfolio

A modern, minimal multi-page portfolio built with React, Tailwind CSS, Framer Motion, and Lucide icons. Warm white luxury aesthetic with deep accent buttons.

## Pages

- **Home** — Hero, About, Skills, Education, Featured Projects, Social links
- **Projects** — Full project grid
- **Resume** — Preview, download, experience, certifications
- **Contact** — Contact form and social links

## Color Palette

| Token | Hex |
|-------|-----|
| Warm background | `#F9F1E8` |
| Secondary white | `#FFFDF9` |
| Primary text | `#1E1E1E` |
| Secondary text | `#666666` |
| Accent / buttons | `#280807` |
| Borders | `#E8DED4` |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Customize

Edit files in `src/data/` for profile, projects, skills, education, and certifications.

Add your resume to `public/resume.pdf`.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React + Vite
- Tailwind CSS v4 (`@tailwindcss/vite`)
- React Router DOM
- Framer Motion
- Lucide React
